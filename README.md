# MINFLIX Project

- 설명: 영화 API를 이용한 영화 사이트
- [화면이동](https://azurealstn.github.io/minflix/)

## ✨ 기능

- 네이게이션 만들기
- YTS 영화 API (axios 통신)
- axios는 네트워크를 통해서 속도가 느리기 때문에 비동기적 처리 (async & await)
- loading 구현
